# wxext-ui

A set of front -end management platform based on Ant Design

## Usage

```sh
# clone the project
git clone git@192.168.10.71:others/wxext-ui.git

# enter the project directory
cd wxext-ui

# set registry

npm config set registry https://registry.npmmirror.com/

# install dependency
npm install

# develop
npm start
```
